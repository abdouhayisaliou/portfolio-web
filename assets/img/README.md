# Images du portfolio

## Photo de profil
- Fichier : `profile.jpg`
- Page : À propos
- Remplacez par votre photo professionnelle (ratio 4:5, min. 800×1000 px).

## Captures de projets
- Dossier : `projects/`
- Nommez chaque fichier comme le slug du projet : `nedd-ong.jpg`, `sos-toitures.jpg`, etc.
- Format recommandé : JPG ou WebP, 1280×820 px minimum (capture d’écran du site réel).

## Régénérer les miniatures temporaires
```bash
python scripts/generate_placeholders.py
```

## Hero accueil
- Optionnel : `hero-realisations.jpg` pour une image hero dédiée.
